== Description ==

Experience seamless redirects with our plugin!

**Intuitive Setup.** Redirects made easy! Effortlessly create and manage redirects with an intuitive interface, ensuring a smooth and user-friendly process.

**Detailed Logs.** Dive into comprehensive logs providing insights into redirect details, including user info, source URL, and destination. Stay informed and in control. (Logs include non-confidential details like device information, date of redirect, etc.)

**404 Error Magic.** Transform 404 errors into opportunities! Instantly redirect them to a specified page, ensuring no visitor gets lost. Effortless navigation, always.

**Config Import/Export.** Seamlessly migrate with ease! Import and export configuration and database files effortlessly, simplifying the process of moving your settings across environments.

== Installation ==

From your WordPress dashboard

1. **Visit** Plugins > Add New
2. **Search** for "Swift Redirect"
3. **Install and Activate** Swift Redirect from your Plugins page
4. **Click** on the new menu item "Swift Redirect" and create your first Redirect!
